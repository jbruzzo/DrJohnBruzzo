# Acerca de Este Blog

Soy un ingeniero mecánico especializado en **análisis de fatiga**, **dinámica estructural** y **sistemas de bombeo**.

Este blog comparte:
- Metodología de diseño a fatiga
- Herramientas computacionales open-source
- Análisis de casos reales de ingeniería
- Tutoriales de Python para ingenieros

## Herramientas que Usamos

- **pyLife** - Análisis de fatiga con curvas de Wöhler
- **py_fatigue** - Rainflow counting y Paris law
- **Python** - NumPy, Matplotlib, Jupyter
- **Open Source** - Todo reproducible y descargable
